# Comment s'organiser en groupe pour la réalisation de projet file rouge

Organisation de groupe Lead Dev pour le projet file rouge

## Objectifs de formateur 

tous les membre de groupe, doit passer par les troix catégories des tâches 

1. Besoin
2. Analyse techniquer et labs
2. Analyse fonctionnelle
3. Conception 
  - Diagramme de casse
  - Maquettage
4. Réalisation 

ce sont des phase théorique on pratique pour chaque tâche l'apprenant doit passer par 

1. Travail à faire 
2. labs
3. Fonctionnaluté à développer 
   1. Diagamme de cas d'utilisation
4. Diagramme de classes
5. Maquette à développer
6. Réalisation 
   1. description des classes service à créer 
   2. création de la base de donnée
   3. les jeux de test


## Organisation de dépôt de travail 

### Branche main 

contient le code finale à déployer dans le serveur.


### branche develop 

branche de développement , tous les appreant doit merger leur travail dans cette branche.

### branche individuelle 

chaque apprenant doit créer une branche individuelle qui port son nom 

exemple : 

madani.ali

il doit contient la strcutre des sous dossier 

- 01_fonctionnalité_a_developper
- 02_labs
- 03_maquette
- 04_realisation
- README.md

dans chaque dossier doit contient un fichier README.md qui contient la description de la tâche.

## madani.ali  : README.md

Description de travail à faire : résumé : 

- Fonctionnalité à développer : Cas d'utiliusation
- labs à réaliser : liste des labs à réaliser 
- maquette à développer : liste des maquette à développer
- réalisation :  tâches à faire pendant la réalisation 
    - par exemple : un apprenat qui qui réaliser la base de donnée
    - un appprenant qui a réalser les jeux de test
    - description des classes service à créer avec leurs méthode 
      - l'apprenant doit noter seulement les méthoque qu'il a besoin à créer 
  - on note dans : Réaliser les tâche à réaliser dans la phase de réalisation
  - la conception technique qui n'est pas encord fait dans la pahse de conception comme la conception de la couche : Service.
